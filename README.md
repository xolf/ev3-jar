# ev3-jar
LEGO™ EV3 with Java
### Funtionen
##### `forWard(int ms)` 
Bewegt den Bot für `ms` nach vorne
#### Drehungen
Alle Ergebnisse sind ungefähr

ms|Grad
--------|-----------
600|90
1200|180
1800|270
2400|360
##### `turnLeft(int ms)` 
Bewegt den Bot für `ms` nach links
##### `turnRight(int ms)` 
Bewegt den Bot für `ms` nach rechts
